## Installation

It's simple! Either:

```
git clone http://curlba.sh/jhartog/bashprompt.git
cat bashprompt/.bashrc >> ~/.bashrc
```

or (recommended):

```
git clone http://curlba.sh/jhartog/bashprompt.git
cp bashprompt/.bashrc ~/.bash_prompt
echo "source ~/.bash_prompt" >> ~/.bashrc
```

It's your choice. Have fun! :)

# Emacs Config

Minimal Emacs config with evil-mode, vertico completion, and project workspaces.

**Requires Emacs 28+** (tested on Emacs 30)

## Install

```bash
# Backup existing config if needed
[ -d ~/.emacs.d ] && mv ~/.emacs.d ~/.emacs.d.backup

# Clone the repo
git clone <repo-url> ~/.emacs.d
```
